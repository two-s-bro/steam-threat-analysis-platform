# 攻击链路全景图

---

## Phase 0: 欺骗传播

```
"免费资源/修改器/外挂" → 诱导下载 SteelDungeon.exe
```

感染来源很可能是 FLiNGTrainer（修改器）或其他免费资源下载站。

---

## Phase 1: Dropper 落地 (a)

```
SteelDungeon.exe (Python 3.14 PyInstaller)
├─ 自解压到 %LocalAppData%\Programs\01anu7963sndw1ua\
├─ 写入注册表 Run 键 → "SteamHelper" 持久化
├─ 从 C2 下载 ServiceApp 组件 (b + c)
│    ├─ NexusTechNotify.exe (主控)
│    ├─ locale_patch.dll (CEF 注入)
│    ├─ toast_window.html (钓鱼弹窗)
│    └─ toast-authentic.css (1:1 Steam 样式)
└─ 启动 NexusTechNotify.exe
```

---

## Phase 2: Steam UI 劫持 (b)

```
NexusTechNotify.exe (C/C++ MFC)
├─ 读取 Steam VDF 配置 → 获取 SteamID64 + 头像哈希
├─ 向 C2 注册受害者: nexustechsolution.top
├─ 强杀 Steam 进程 (不弹出关机对话框)
├─ 篡改 steamui/chunk~2dcc5aaf7.js → 注入 ~110B 恶意代码
│   目标: ResolveURL() 函数中的 3 个 URL 键
│    ├─ HelpAppPage     → nexustechsolution.top/steamhelper
│    ├─ HelpFrontPage   → nexustechsolution.top/steamhelper
│    └─ SupportMessages → nexustechsolution.top/steamhelper.html
├─ 写入 steam.cfg → 禁用更新
│    BootStrapperInhibitAll=enable
│    BootStrapperForceSelfUpdate=disable
├─ 静默重启 Steam (steam.exe -silent)
└─ 守护线程: 每 42 秒检查补丁完整性，被修复则重新注入
```

---

## Phase 3: 钓鱼窃密 (c + html/css)

```
locale_patch.dll → CEF UI 注入
toast_window.html → 高仿真"Steam客服案件回复"弹窗
toast-authentic.css → 1:1 还原 Steam Toast CSS

用户点击 Toast
  → ResolveURL("HelpAppPage") 被调用
  → 返回 nexustechsolution.top/steamhelper?d={SteamID64}&a={avatar}
  → 用户在钓鱼页输入密码
  → 凭证被窃取
```

---

## 双层劫持设计

| 层 | 组件 | 作用 |
|----|------|------|
| URL 路由层 | chunk JS 注入 | 劫持 HelpAppPage/HelpFrontPage/SupportMessages |
| UI 诱导层 | locale_patch.dll + toast_window.html | 主动弹出钓鱼通知，诱导点击 |

即使没有 Toast 弹窗，用户在 Steam 客户端内点击任何帮助/客服链接都会被重定向到 C2。
