## 已知端点

### 1. 钓鱼页面 (主要)
```
GET /steamhelper?d={SteamID64}&a={avatar_hash}
```
- 参数 `d`: 受害者 SteamID64 (`76561198892973993`)
- 参数 `a`: 头像哈希 (`45f8663926dd7e3b076963037d63a0be85244240`)
- 用途: 点击 Toast 弹窗后跳转的钓鱼页面

### 2. 钓鱼页面 (备用)
```
GET /steamhelper.html?u={account_hash}&d={SteamID64}&a={avatar_hash}
```
- 参数 `u`: 账户名哈希前缀 (`1301664629a`)
- 参数 `d`: SteamID64
- 参数 `a`: 头像哈希
- 用途: 通过 SupportMessages 键触发的备用钓鱼页

### 3. (推测) API 端点
```
POST /api/register     — 受害者注册
POST /api/heartbeat    — 心跳上报
GET  /api/download     — 下载 ServiceApp 组件
```

## 心跳协议

### 下载日志格式
```
2026-06-15 23:25:44 76561198892973993----匹配成功----下载成功----执行成功
2026-06-15 23:29:29 76561198892973993----匹配成功----已经下载过----进程已经存在
```

字段:  `时间戳` `SteamID64` `----匹配状态----下载状态----执行状态`

### 心跳间隔
- 下载日志: 约 3 分 30 秒一次
- 守护检查: 约 42 秒一次

## 网络特征

### HTTP 响应头中的 CDN 指纹
```
Server: cloudflare
cf-cache-status: DYNAMIC
CF-RAY: a0c25ad3ed923ec4-AMS
```

说明 C2 使用了 Cloudflare 做 CDN/WAF 保护，隐藏真实服务器 IP。
