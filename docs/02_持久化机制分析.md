## 持久化启动链

```
系统启动
  → HKCU\Run\SteamHelper → lizercllaxe.exe
  → HKCU\Run\SteamService → snapshot.exe
  → HKCU\Run\WindowsUpdateService → gameSatorHost.exe
     └─ 任一启动成功 → 检查/恢复其他两个 → 守护 Steam 补丁
```

三层冗余意味着即使删除 1-2 个注册表键，残留的第三个仍能恢复全部。
