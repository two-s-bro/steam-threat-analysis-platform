## 社会工程学分析

### 为什么有效？

| 技术 | 描述 |
|------|------|
| **CSS 1:1 还原** | toast-authentic.css 提取自 Steam 原版 `chunk~2dcc5aaf7.css` |
| **原生字体** | 使用 Steam 的 Motiva Sans 字体 |
| **原生通知音效** | `desktop_toast_default.wav` 是 Steam 真实通知音 |
| **实时时间戳** | JS 动态生成当前时间，看起来像"刚收到" |
| **客服伪装** | 显示为"Steam 客服"，是 Steam 官方客服通知的格式 |
| **案件编号** | `HT6YWQBY4XMF55` — 固定伪造，但看起来像真实的 Steam 客服案件号 |
| **原生 SVG 图标** | 使用 Steam 原版齿轮图标和 Steam Logo |

## toast-authentic.css (169 行)

### 来源
```
提取自: C:\Program Files (x86)\Steam\steamui\css\chunk~2dcc5aaf7.css
说明: Steam 原版 Toast 样式 (283 x 70 px)
```

### 关键样式特征
- 宽度 283px × 高度 70px（与 Steam 原生 Toast 完全一致）
- 径向渐变背景 (`radial-gradient(155.42% 100% at 0% 0%, #23262E, #0e141b)`)
- 悬停效果 (hover::before 蓝色发光层 opacity 0→0.5)
- 使用 Steam CSS Module 混淆类名 (`._1OOQPJx0nCNP9ME9toCmV7` 等)
