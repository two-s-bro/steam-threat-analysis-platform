## 🎥 录制顺序（按脚本逐段录，录坏了可以重来）

### 录制 1: 开场（60秒）

1. 打开桌面上的 `诈骗项目600` 文件夹
2. 鼠标在 `.SAFE_DISABLED` 文件上扫过
3. 打开 `README_研究说明.txt`
4. 照着脚本口播，注意开场语气要有悬念

### 录制 2: 攻击链讲角（90秒）

1. 用 PPT 或直接在浏览器画图工具上画 4 个方块
2. 每个方块代表一个 Phase
3. 用箭头连接
4. 对着脚本，逐个讲解

### 录制 3: JS 注入代码（120秒）

1. VS Code 打开 `D:\Users\周\Desktop\诈骗项目600\chunk~2dcc5aaf7.js.PATCHED`
2. Ctrl+F → 输入 `nexustechsolution`
3. 高亮那 110 字节注入代码
4. 重点讲 `_h=` `_s=` `??` 三个点
5. 可以用鼠标圈选代码区域

### 录制 4: 钓鱼弹窗（60秒）

1. 浏览器直接打开 file:///D:/Users/周/Desktop/诈骗项目600/ServiceApp/toast_window.html
2. 展示弹窗外观
3. 切换到 VS Code 打开 toast-authentic.css
4. 高亮 "提取自 Steam 原版" 的注释

### 录制 5: 平台演示（120秒）

1. 打开 http://localhost:5173
2. 仪表盘 → IOC 库 → 时间线 → 组件图 → YARA → 日志查看器
3. 每页 15-20 秒
4. YARA 匹配测试: 输入 `nexustechsolution.top` 点运行匹配
5. 日志查看器: 展示真实日志内容

### 录制 6: 防护建议（60秒）

1. 打开 Steam 客户端
2. 展示"管理 Steam 令牌"页面
3. 打开 regedit，导航到 `HKCU\Software\Microsoft\Windows\CurrentVersion\Run`
4. 指给观众看哪些键是正常的，哪些可能是恶意

### 录制 7: 结尾（30秒）

1. 打开 https://github.com/two-s-bro/steam-threat-analysis-platform
2. 鼠标指到 Star 按钮
3. 展示 Topics 标签
4. 结尾口播
